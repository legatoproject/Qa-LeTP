#########################
Tests for services
#########################

These are the tests for the services of Legato Application Framework.
They are divided into two categories:

1. Tests for the secure storage.
2. Tests for the updating functionality. Update tests are also divided into two:
      1. Tests for the flash APIs
      2. Tests for the update control API
