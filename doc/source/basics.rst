#########################
Tests for basic functions
#########################

These are the tests for the basic functions of Legato Application Framework.
They are divided into three categories:

1. Set of tests for the Legato component and application definition files.
2. Set of tests for the Legato sample apps.
3. Set of tests for the Legato kmod tools
